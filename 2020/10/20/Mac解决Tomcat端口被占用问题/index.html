Mac解决Tomcat端口被占用问题

打开终端，运行：

  1. lsof -i:<端口号>
    实例：lsof -i:8080
    则会显示比如：
    COMMAND PID USER FD TYPE DEVICE SIZE/OFF NODE NAME
    java 32841 sss 43u IPv6 0x3008a38c418a6377 0t0 TCP *:http-a

  2. 接着输入：
    命令：kill -9 <进程ID>
    实例：kill -9 48168
